### 项目简介：

关键词：咖啡；新零售；XX；XX；XX

摘要：我们这个时代一个新品牌蹿红的速度越来越短，最近不管是地铁还是电梯口，张震和汤唯无处不在的广告语“这一杯，谁不爱”占满大街小巷（Ps：汤唯真女神，美的不食人间烟火，只喝咖啡。）
