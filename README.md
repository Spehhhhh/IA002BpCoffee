## 项目简介

**关键词：**

咖啡；新零售；消费升级；

**摘要：**

我们这个时代一个新品牌蹿红的速度越来越短，最近不管是地铁还是电梯口，张震和汤唯无处不在的广告语“这一杯，谁不爱”占满大街小巷（Ps：汤唯真女神，美的不食人间烟火，只喝咖啡。）

这个小蓝杯是如何在巨头“星巴克”的眼皮底下，一年时间迅速走红，成为中国2018年万马齐喑，市场萎靡的创业大军中一匹现象级的黑马，我们能从这个年轻的新品牌当中学到什么？它和星巴克的差异化在哪？它未来能否撼动老大哥星巴克？
